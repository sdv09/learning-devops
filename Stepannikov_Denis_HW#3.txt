Задание 1.
Кейс:
Компания ООО "Рога и Копыта" решила автоматизировать свою работу и централизировать управление учетными записями. Руководством было принято решение развернуть пять новых серверов:
	1. систему контроля доступа пользователей и управления учетными записями;
	2. терминальный сервер;
	3. два сервера для веб-приложений и прокси сервер.
Какие ОС вы выберете для каких серверов? Почему?

Решение:
	1. В качестве системы контроля доступа доступа пользователями выберу Active Directory (windows) либо FreeIPA (CentOS). в текущей практике для WIN окружения используем AD в Linux - FreeIPA.  
	2. В качестве терминального сервера буду использовать семейство Windows c ролью терминального сервера. пока для меня это максимально удобрый вариант, который закрывает все потребности.
	3. Для веб сервера Windows IIS/nginx в качестве прокси сервера - Nginx. в работе использую IIS (на OC Windows) и nginx (на ОС Centos). nginx (бесплатная версия) не поддерживает ntlm авторизацию, в т.ч. по этому есть пару серверов на IIS.   

В общем в качестве ОС Linux семейства используем пока CentOS7, пытаемся перейти на Ubuntu, но большая часть средств автоматизации написаны пока только для CentOS.
